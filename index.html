סדנת הפוסטר ב-GIMP

בסדנה הזו תבנה פוסטר מלא - מקנבס ריק ועד קובץ מוכן לייצוא - בעזרת GIMP, תוכנת עריכה חינמית וחזקה. עבור על התחנות אחת-אחת, ותסמן V כשסיימת כל שלב. תקוע? טוביה תמיד זמין בכפתור למטה.

▶ צפה בסרטון שממחיש את כל השלבים לפני שמתחילים, או תוך כדי עבודה - כדאי לצפות בהדגמה מלאה של התהליך ב-YouTube.
פתיחת קנבס

פתיחת קנבס חדש

  1. פתח את GIMP ובחר File ← New.
  2. הגדר גודל קנבס לפוסטר, למשל 2480×3508px (גודל A4 להדפסה).
  3. ודא ש-Resolution מוגדר על 300 DPI אם מתכננים להדפיס.
  4. שמור מייד את הקובץ בפורמט .xcf (הפורמט המקורי של GIMP), כדי לשמר שכבות.
בניית הרקע

בניית הרקע

  1. צור שכבה חדשה (Layer ← New Layer) וקרא לה Background.
  2. בחר צבע אחיד עם כלי Bucket Fill, או צור מעבר צבעים (Gradient) עם כלי Blend Tool.
  3. אפשר גם לייבא תמונה כרקע: File ← Open as Layers.
  4. ודא שהרקע נמצא בשכבה הכי תחתונה בחלונית Layers.
כותרת וטקסט

הוספת כותרת וטקסט

  1. בחר את כלי Text Tool (T) ולחץ על הקנבס להוספת כותרת ראשית.
  2. בחר גופן ברור וקריא לכותרת, וגודל גדול משמעותית מכל טקסט אחר בפוסטר.
  3. הוסף שכבת טקסט נוספת לפרטים משניים (תאריך, מיקום, קרדיט וכו').
  4. שמור על ניגודיות טובה בין צבע הטקסט לרקע - אם קשה לקרוא, שקול להוסיף מאחורי הטקסט תיבה שקופה למחצה.
אלמנטים גרפיים

הוספת אלמנטים גרפיים

  1. הוסף תמונה, איור או צורה מרכזית עם File ← Open as Layers, או צייר צורות עם כלי Paths / Shapes.
  2. אם התמונה כוללת רקע לא רצוי - השתמש בכלי Select by Color או Fuzzy Select להסרתו, או ב-Layer Mask לשקיפות עדינה.
  3. סדר את השכבות בחלונית Layers כך שהכותרת תמיד תישאר קריאה מעל שאר האלמנטים.
אפקטים וגימור

אפקטים וגימור

  1. נסה פילטר עדין מתפריט Filters - למשל Light and Shadow ← Drop Shadow לטקסט או לצורה מרכזית.
  2. שחק עם Layer Modes (למשל Multiply / Screen) לשילוב מעניין בין שכבות.
  3. עבור על כל השכבות ווודא ניגודיות, יישור, ואיזון כללי - התרחק מהמסך רגע ותסתכל על הפוסטר בעין "טרייה".
ייצוא הפוסטר

ייצוא הפוסטר הסופי

  1. שמור קודם גרסת עבודה: File ← Save (שומר .xcf עם כל השכבות).
  2. לייצוא סופי: File ← Export As ← בחר .png (איכות מלאה) או .jpg (קובץ קטן יותר).
  3. צלם מסך של תהליך העבודה (השכבות, הכלים ששימשו) - תזדקק לזה בהמשך למסמך התיעוד של מטלת הביצוע.
